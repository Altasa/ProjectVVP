Файл receips содержит функцию обновления и записи в файл ./Data/receips.dat словарей
имен подкатегорий, названий и адресов всех рецептов

В файле proreceip функция поиска конкретного рецепта - принимает url рецепта
Возвращает словарь с рецептом
Шаблон вызова функции поиска в файле example_fun

В файле rsauce описание словаря рецепта, а также извлечение словарей имен из файла

Некоторые категории на сайте либо имеют менее 15 рецептов, либо пусты. 
Эти случаи вызывают словарь рецепта UNKNOWN URL, а также блокируют отображение кнопок

Также существует проблема совместной работы многих пользователей, так как
при одновременных запросах пользователи обращаются к одному и тому же объекту
словаря рецепта, и один пользователь перезаписывает данные другого
Необходимо либо держать копию выбранного рецепта для каждого пользователя,
либо сделать получение компонентов рецепта также асинхронным

Для обновления (создания) файла имен и адресов receips.dat
запустите файл receips (файл данных создается в папке Data в корне рабочей директории, заполняется ~5 минут)
файл secret_info содержит строку TOKEN='токен собственно'

библиотеки
python-telegram-bot 20.0a6
lxml 4.9.0 (установите файл через pip), beautifulsoup4, requests, pickle
